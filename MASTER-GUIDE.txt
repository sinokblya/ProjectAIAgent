🎯 СЛЕДУЮЩИЕ ШАГИ:

1️⃣  Скопируйте все файлы в папку edagent/

2️⃣  Отредактируйте .env:
    $ cp .env.example .env
    $ nano .env  # Добавьте API ключи

3️⃣  Запустите приложение:
    $ docker-compose up -d

4️⃣  Откройте в браузере:
    http://localhost

5️⃣  Используйте приложение:
    - Пройдите через 3 фазы
    - Посмотрите API документацию
    - Настройте облачный хостинг
